## Installation
cd to project directory
Use the package manager [npm](https://nodejs.org/en/download/) to install modules.

```bash
npm i express
npm i -g nodemon
npm i ejs
npm i mongoose
```

To Run:
```bash
nodemon app.js
```

To Run DB:
```bash
mongod
```
Create a new bash tab:
```bash
mongo
show dbs
```

currently, this app is unable to remove items. the "X" button is just for show.
